## Roles in CampusOps

1. Admin
- Creates users
- Assigns roles
- Manages courses and subjects

2. Faculty
- Creates lesson plans
- Creates assignments

3. Student
- Views courses
- Submits assignments

## AI Feature

Faculty can use an AI tool to generate lesson plans.

The faculty enters:
- Subject name
- Topic
- Duration

The AI returns:
- A structured lesson plan

Faculty can edit the result before saving.
