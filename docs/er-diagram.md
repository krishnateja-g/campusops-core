## Data stored in CampusOps

1. User
- Name
- Email
- Password
- Role (Admin / Faculty / Student)

2. Course
- Course name
- Course code

3. Subject
- Subject name
- Belongs to a course

4. Assignment
- Title
- Description
- Due date
- Belongs to a subject

5. Submission
- Assignment
- Student
- Submitted file or text
